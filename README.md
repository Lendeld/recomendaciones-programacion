# 📚 Recomendaciones y Notas

Repositorio público con recomendaciones, guías y buenas prácticas de desarrollo de software.
Incluye ejemplos, configuraciones y tips basados en experiencia real en proyectos modernos.

---

## 🚀 Objetivo

Centralizar conocimiento práctico sobre:

* Buenas prácticas de programación
* Arquitectura de software
* Seguridad (manejo de secretos, variables de entorno)
* Configuración de entornos
* Herramientas de desarrollo
* Tips para productividad

Este repositorio está pensado como una referencia rápida tanto para uso personal como para otros desarrolladores.

---

### Ejemplos:

* 🔐 Manejo seguro de `.env` y `appsettings.json`

---

## 🔐 Seguridad

Algunas reglas importantes incluidas en este repo:

* Nunca subir archivos sensibles:

  * `.env`
  * `appsettings.*.json` con secretos
* Separar configuración por entorno
* Uso de servicios como:

  * Azure Key Vault
  * Variables de entorno del sistema

---

## 🧠 Filosofía

* Código limpio > código rápido
* Seguridad desde el inicio
* Simplicidad antes que complejidad
* Automatizar lo repetitivo
* Documentar lo importante
